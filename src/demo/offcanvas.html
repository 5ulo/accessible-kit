Usage

Basic HTML Structure:

<!-- Trigger button -->
<button data-offcanvas-trigger="my-offcanvas">Open</button>

<!-- Offcanvas -->
<div data-offcanvas id="my-offcanvas">
    <div data-position="left" data-offcanvas-panel>
        <div data-offcanvas-header>
            <h2 data-offcanvas-title>Title</h2>
            <button data-offcanvas-close aria-label="Close">×</button>
        </div>

        <div data-offcanvas-body>
            Offcanvas content...
        </div>
    </div>

    <div data-offcanvas-backdrop></div>
</div>

JavaScript Initialization:

1. Auto-initialization (CDN)

<script src="js/a11y-offcanvas.js"></script>
<!-- Automatically initializes all offcanvas -->

2. CDN with window.a11yKit namespace (recommended)

<script src="js/a11y-offcanvas.js"></script>
<script>
  // Init function
  a11yKit.initOffcanvas();

  // Bootstrap style
  const offcanvas = new a11yKit.Offcanvas(element, {
    closeOnBackdrop: true,
    onOpen: (instance) => console.log('Opened')
  });
</script>

3. ES6 Module Import - Bootstrap Style

import { Offcanvas } from './a11y-kit/a11y-offcanvas.js';

const offcanvas = new Offcanvas(element, {
    closeOnBackdrop: true,
    closeOnEscape: true,
    trapFocus: true,
    scrollLock: true,
    onOpen: (instance) => console.log('Opened'),
    onClose: (instance) => console.log('Closed')
});

4. Wildcard / NPM Package

// Wildcard
import * as a11yKit from './a11y-kit';
const offcanvas = new a11yKit.Offcanvas(element, options);

// NPM
import { initOffcanvas, Offcanvas } from 'accessible-kit/offcanvas';
initOffcanvas(); // or new Offcanvas(element, options);

Keyboard Navigation

  • Tab - Navigate between focusable elements (with focus trap)
  • Shift + Tab - Reverse navigation (with focus trap)
  • Esc - Close offcanvas and return focus to trigger

Features

  • ✅ Full ARIA attributes support (role="dialog", aria-modal="true")
  • ✅ Focus trap - Tab cycles only through elements in offcanvas
  • ✅ Return focus - Returns focus to trigger after closing
  • ✅ Scroll lock - Locks body scrolling
  • ✅ Escape to close
  • ✅ Backdrop click to close
  • ✅ 4 positions (left, right, top, bottom)
  • ✅ Configurable behavior
  • ✅ Responsive design
  • ✅ Dark mode support
  • ✅ Reduced motion support
